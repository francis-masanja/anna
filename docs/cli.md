# Command-Line Interface

Anna AI provides a command-line interface for interacting with the application.

## Usage

```
julia main.jl [command] [options]
```

## Commands

### `interactive`

Starts Anna AI in interactive mode.

## Options

### `--env <environment>`

Specifies the environment to use. The possible values are:

- `development` (default)
- `staging`
- `production`
